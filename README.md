# MovieGo Usability Testing & Analysis

## Project Overview

This project evaluates the usability of MovieGo, a mobile application prototype for online movie ticket booking. Usability testing was conducted with sample users to identify navigation issues, task completion challenges, and areas for user experience improvement.

## Objectives

* Conduct usability testing on the MovieGo prototype
* Observe user interactions and behaviors
* Collect feedback from participants
* Analyze usability issues
* Recommend design improvements

## Test Participants

* 5 participants
* Age Group: 18–35 years
* Mixed experience with online ticket booking apps

## Test Tasks

1. Login to the application
2. Search for a movie
3. Select a theatre
4. Choose seats
5. Complete payment
6. View booking confirmation

## Usability Metrics

* Task Completion Rate
* Time on Task
* Number of Errors
* User Satisfaction Score

## Findings

* Search option was difficult to locate.
* Users hesitated during seat selection.
* Payment button visibility was low.
* Confirmation page lacked booking details.

## Recommendations

* Increase visibility of search functionality.
* Improve seat availability indicators.
* Highlight payment actions.
* Add detailed booking information on confirmation screens.

## Tools Used

* Figma
* Microsoft Excel
* Google Forms

## Outcome

The project demonstrates the complete usability testing process, from planning and user observation to analysis and design improvement recommendations.
